# Open Company Research

Project Rules

## Core Principle

Separate Engineering from Content.

Engineering never writes educational content.

## Claude Code Responsibilities

Responsible ONLY for:

- Project architecture
- Folder structure
- Markdown formatting
- Git
- MkDocs
- Mermaid validation
- Navigation
- Templates
- GitHub Actions
- Refactoring
- Documentation infrastructure

Claude Code MUST NOT write:

- Book chapters
- Tutorials
- Company analysis
- Industry analysis
- Research reports
- Investment opinions
- Learning materials
- Educational content

Empty markdown files remain:

```markdown
# Title

TODO
```

Nothing more.

## ChatGPT Responsibilities

Responsible for:

- Book writing
- Knowledge architecture
- Learning path
- Company research
- Industry research
- Mermaid diagrams
- Exercises
- Case studies
- Reading guides
- Chapter writing

Only ChatGPT writes educational content.

One chapter

One Commit

One Release

## Content Policy

Never automatically fill TODO.

Never generate book content.

Wait for manual content.
