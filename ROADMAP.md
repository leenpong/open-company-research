# Roadmap

本文件描述 **Open Company Research** 的长期演进规划。版本遵循
[语义化版本](https://semver.org/lang/zh-CN/)，里程碑按内容成熟度而非时间划分。

> 状态图例：⬜ 未开始 · 🟡 进行中 · ✅ 已完成

---

## Version 0.1 — 项目初始化

**目标：搭建一个 Production Ready 的空壳，让后续内容可以顺畅注入。**

- ✅ 初始化目录结构与工具链。
- ✅ 根级文档（README / LICENSE / CONTRIBUTING 等）。
- ✅ MkDocs Material 配置与本地预览。
- ✅ GitHub Actions：Lint / Link Check / Spell Check / Build。
- ✅ 写作规范（STYLE_GUIDE）与模板骨架。

## Version 0.5 — 基础方法论

**目标：让读者掌握「如何研究一家公司」的通用方法。**

- ⬜ 完成 `01-fundamentals` 全部章节（财报、商业模式、护城河、估值）。
- ⬜ 完成 `06-templates` 模板说明与使用示例。
- ⬜ 建立 `glossary` 核心术语表。
- ⬜ 首批 Mermaid 图与思维导图。

## Version 1.0 — 首个完整专题与案例

**目标：交付端到端可用的知识库首个正式版本。**

- ⬜ 完成一个完整行业专题（`02-ai-industry` 优先）。
- ⬜ 完成一个完整实战案例（`05-case-study`）。
- ⬜ 上线 GitHub Pages（启用 Deploy 工作流）。
- ⬜ 补全首页与学习路线。

## Version 2.0 — 多行业与可视化体系

**目标：从「单点深入」走向「体系覆盖」。**

- ⬜ 完成 `03-semiconductor`、`04-cloud` 专题。
- ⬜ 建立统一的可视化体系（时间线、对比图、产业链图谱 `10-industry-maps`）。
- ⬜ 完善 `07-learning` 学习资源（Books / Courses / Videos / Papers）。
- ⬜ 建立 `09-companies` 公司研究档案，沉淀 10+ 完整案例。

## Version 3.0 — 社区规模化

**目标：成为中文公司研究的事实标准知识库。**

- ⬜ 建立成熟的社区共建与评审机制。
- ⬜ 多语言 / 多版本支持探索。
- ⬜ 300+ Markdown、30~50 万字规模。
- ⬜ 与外部数据源 / 工具生态联动。
