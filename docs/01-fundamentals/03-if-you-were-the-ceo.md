# 第三章｜如果明天你成为 NVIDIA CEO，你今天最关心什么？

很多投资者进入市场后的第一件事，就是学习如何看 K 线、研究各种技术指标、寻找买卖点。他们相信，只要能够更准确地预测价格，就能获得更好的投资收益。

然而，一个有趣的问题是：如果明天早上，你不再是一名投资者，而是 NVIDIA 的 CEO，你今天还会把时间花在研究股价上吗？

答案几乎是否定的。

假设明天上午九点，你需要主持董事会会议。会议室里坐着公司的高管、董事和核心业务负责人。会议开始后的第一个小时，没有人会讨论今天股价上涨了多少，也不会有人询问 MACD 是否形成金叉，更不会有人分析短期市场情绪。

所有人的注意力都会集中在真正影响公司未来的事情上。

例如，今年的数据中心业务还能保持高速增长吗？竞争对手有没有推出足以威胁公司领先地位的新产品？下一代 GPU 是否能够按计划交付？最大的云计算客户是否还会继续扩大采购规模？未来三年的研发重点应该放在哪里？

这些问题看似与股票无关，却决定了股票长期的价值。

很多人误以为，优秀的投资者之所以能够获得更好的收益，是因为他们更擅长预测市场。事实上，长期投资者真正擅长的，并不是预测价格，而是理解企业。

股价每天都会变化，而一家企业真正重要的东西却变化得很慢。

品牌需要很多年才能建立。

技术优势需要持续投入研发才能形成。

客户关系需要长期积累。

企业文化、管理团队、供应链和商业生态，更不是几个月就能复制出来的。

这些因素，才是真正决定企业价值的核心。

遗憾的是，大多数投资者每天关注的，却是最容易变化、也是最容易干扰判断的价格。

如果把视角放到一家普通餐厅，这个道理会更加容易理解。

假设这家餐厅属于你。每天晚上关门之后，你最关心的事情是什么？

你不会坐在那里思考："如果今天有人收购我的餐厅，它值多少钱？"

你真正关心的是：今天来了多少客人？回头客有没有增加？哪些菜最受欢迎？有没有收到新的差评？厨房成本有没有继续上涨？主厨是否准备离职？附近有没有新的竞争对手即将开业？

因为你知道，这些问题决定的是餐厅未来还能不能赚钱。

如果这些经营指标越来越好，那么即使暂时没有人愿意高价收购，你依然拥有一家越来越优秀的企业；反过来，如果这些问题不断恶化，即使今天有人愿意给出不错的价格，这种价值也很难长期维持。

上市公司其实也是一样。

只是餐厅变成了几万名员工的大型企业，菜单变成了产品线，顾客变成了全球客户，厨房变成了研发中心，而每天营业额变成了财务报表上的收入和利润。

经营企业的底层逻辑，从来没有改变。

因此，当我们研究一家上市公司时，不妨试着把自己的身份暂时从投资者切换成经营者。

不要先问："这家公司还能买吗？"

先问："如果我是这家公司的 CEO，我现在最担心什么？"

也许你会担心核心客户流失，也许你会担心竞争对手推出了更先进的产品，也许你会担心供应链出现问题，也许你会担心研发投入不足导致未来失去竞争优势。

当你开始认真思考这些问题时，你会发现，自己的关注点已经发生了变化。

你不再只是观察价格，而是在理解企业。

这种思维方式的改变，看似简单，却是普通投资者和长期投资者之间最重要的区别之一。

真正优秀的投资者，几乎都会不断思考同样几个问题：

- 这家公司真正为客户创造了什么价值？
- 为什么客户愿意持续选择它，而不是竞争对手？
- 它最大的竞争优势来自哪里？
- 五年以后，这种竞争优势是否依然存在？
- 管理层是否有能力让企业持续成长？

这些问题没有标准答案，也不会在任何行情软件里直接显示。

但恰恰是这些问题，决定了一家企业未来十年的价值。

所以，从这一章开始，希望你逐渐养成一个新的习惯。

每当准备研究一家上市公司时，不要急着打开行情软件，而是先把自己放到 CEO 的位置，认真思考：如果今天真正由我来经营这家公司，我最先需要解决的问题是什么？

当你能够越来越自然地从经营者的角度理解企业，而不是从交易者的角度理解价格时，你的投资研究就已经迈出了最重要的一步。

## 思考题

1. 选择一家你熟悉的上市公司，假设你今天成为它的 CEO，你认为未来三年最重要的战略任务是什么？
2. 如果竞争对手推出一款性能更好、价格更低的产品，你会如何应对？
3. 公司目前最大的竞争优势是什么？这种优势是否能够持续五年以上？
4. 如果只能做一件事情来提升公司的长期价值，你会选择什么？为什么？
