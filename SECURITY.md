# 安全策略

## 支持的版本

本项目为文档类知识库。安全修复始终针对 `main` 分支的最新状态发布。

| 版本 | 是否维护 |
| ---- | -------- |
| `main`（最新） | ✅ |
| 历史标签版本 | ❌ |

## 报告漏洞

本仓库主要包含文档内容与文档构建工具链。若你发现以下任一问题，请负责任地
向我们报告：

- 构建 / CI 流程中的安全隐患（如工作流权限、依赖投毒）。
- 依赖项中的已知漏洞。
- 文档站点配置导致的安全风险。
- 仓库中意外泄露的敏感信息（密钥、令牌等）。

### 报告方式

- **优先**：通过 GitHub 的
  [Private Vulnerability Reporting](https://docs.github.com/zh/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
  私密提交。
- 或发送邮件至维护者（请在此处填写联系邮箱）。

请 **不要** 通过公开 Issue 披露安全问题。

### 我们的承诺

- 我们会在收到报告后尽快确认。
- 我们会评估问题并制定修复方案。
- 修复发布后，我们会（在你同意的前提下）致谢报告者。

## 内容与免责

本知识库不涉及任何交易系统或用户数据处理，内容仅用于教育目的，不构成投资建议。
